# RbTags

(It should) Generate ctags files for your ruby project and your bundled gems.

## Installation

Install it using

`gem install rb_tags`

## Requirements

The gem requires that you have

* `find`
* `ctags`

in your path.

## Usage

Cd into your project and simply type (not yet implemented)

`rb_tags`
